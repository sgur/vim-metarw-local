# Requirements

- [metarw](https://github.com/kana/vim-metarw)

# Usage

## List current directory

`:edit local:`

## List home direcotry

`:edit local:~/`

## View a file

`:edit local:~/.vimrc`

# Customization

## Use `local:` scheme for opening directory with `:edit`

`let g:metarw_local_enable_fallback = 1`
